📝 使用方法

1. 各カードの「生成」ボタンをクリックして画像をプレビューします。

2. 「ダウンロード」ボタンをクリックして画像を保存します。

3. ダウンロードした画像を assets/images/ ディレクトリに配置してください。

4. 商品画像は、無料のストックフォトサイト（Unsplash、Pexelsなど）からダウンロードして assets/images/products/ に配置してください。
